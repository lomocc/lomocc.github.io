# Gitlab CI 问题总结

## *docker:dind* ([docker-in-docker](https://blog.docker.com/2013/09/docker-can-now-run-within-docker/)) 缓存镜像

**.gitlab-ci.yml**

```
variables:
  DOCKER_DRIVER: overlay
```

**runner**

```
[runners.docker]
  image = "docker:dind"
  privileged = true
  volumes = ["/cache", "/var/run/docker.sock:/var/run/docker.sock"]
```
